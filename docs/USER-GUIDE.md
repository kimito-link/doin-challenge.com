## 1. はじめに

「動員ちゃれんじ」は、推しの大切なイベントを成功させるために、ファン同士が力を合わせるアプリです。参加表明をすることで、イベント主催者やアーティストに「これだけの人が来る予定です！」という熱意を伝えることができます。

### 主な機能

| 機能 | 説明 |
|------|------|
| チャレンジ参加 | イベントへの参加表明と応援メッセージの投稿 |
| 地域別マップ | 47都道府県からの参加者分布を可視化 |
| 成長軌跡グラフ | 参加者数の推移をリアルタイムで確認 |
| シェア機能 | X（旧Twitter）で参加表明を拡散 |
| バッジ・アチーブメント | 活動に応じた実績バッジを獲得 |

## 3. チャレンジに参加する

### 3.1 チャレンジを探す

ホーム画面には、現在開催中のチャレンジが一覧表示されます。「注目のチャレンジ」セクションには、特に盛り上がっているチャレンジがハイライト表示されます。

**検索方法**

検索バーにキーワードを入力することで、タイトル・ホスト名・会場名でチャレンジを検索できます。

### 3.2 参加表明の手順

1. 参加したいチャレンジカードをタップして詳細画面を開きます
2. 画面をスクロールして「参加表明」セクションを表示します
3. 応援メッセージを入力します（任意）
4. 友人を連れて行く場合は「友人を追加」で登録できます
5. 「応援動画での使用を許可する」にチェックを入れます（任意）
6. 「確認画面へ」をタップして内容を確認します
7. 「参加表明する」をタップして完了です

### 3.3 シェア機能

参加表明が完了すると、X（旧Twitter）でシェアするかどうかを選択できます。シェアすることで、より多くのファンに参加を呼びかけることができます。

## 4. 地域別参加者マップの見方

### 4.1 日本地図ヒートマップ

イベント詳細画面には、47都道府県のデフォルメ日本地図が表示されます。この地図では、各都道府県からの参加者数を視覚的に確認できます。

**色の意味**

| 色 | 参加者数 |
|----|----------|
| 地域カラー（水色・紫・緑など） | 0人（参加者なし） |
| オレンジ | 少数（1〜10%） |
| 赤 | 中程度（10〜40%） |
| 濃い赤 | 多数（40〜70%） |
| 深紅 | 最多（70%以上） |

**アイコンの意味**

| アイコン | 参加者数 |
|----------|----------|
| 😢 | 0人 |
| 😊 | 1〜5人 |
| 🔥 | 6〜20人 |
| 🎉 | 21人以上 |

### 4.2 地域カラー

各地域は固有の色で表示されます。

| 地域 | 色 |
|------|-----|
| 北海道 | 水色 |
| 東北 | 紫 |
| 関東 | 緑 |
| 中部 | 黄色 |
| 関西 | オレンジ |
| 中国 | ピンク |
| 四国 | 紫 |
| 九州 | 赤 |
| 沖縄 | オレンジ |

### 4.3 都道府県の詳細を見る

各都道府県ブロックをタップすると、その都道府県からの参加者一覧がモーダルで表示されます。参加者のプロフィールを確認したり、エールを送ったりすることができます。

## 5. 動員までの軌跡グラフの見方

### 5.1 成長曲線

「動員までの軌跡」グラフでは、参加者数の推移を時系列で確認できます。赤いグラデーションの曲線が参加者数の増加を表しています。

**グラフの要素**

| 要素 | 説明 |
|------|------|
| 赤い曲線 | 参加者数の推移 |
| 緑の点線 | 目標ライン |
| 白い丸マーカー | マイルストーン（25%、50%、75%達成など） |
| 「今」マーカー | 現在の参加者数 |

### 5.2 進捗の確認

グラフ上部には、現在の進捗状況が表示されます。

- **現在の参加者数**: 実際に参加表明した人数
- **目標人数**: チャレンジの目標人数
- **達成率**: 目標に対する達成割合（%）

## 6. チャレンジを作成する

### 6.1 作成手順

1. タブバーの「チャレンジ作成」をタップします
2. チャレンジのタイトルを入力します
3. 目標タイプを選択します（動員/フォロワー/同接/カスタム）
4. 目標人数を設定します
5. カテゴリを選択します
6. イベント日時と会場を設定します
7. チケット情報を入力します（任意）
8. 「作成する」をタップして完了です

### 6.2 目標タイプ

| タイプ | 説明 |
|--------|------|
| 動員 | ライブ・イベントへの来場者数 |
| フォロワー | SNSのフォロワー数 |
| 同接 | 配信の同時接続者数 |
| カスタム | 自由に設定可能 |

## 7. マイページの使い方

### 7.1 プロフィール

マイページでは、自分のプロフィール情報を確認・編集できます。Xアカウントから取得した情報が表示されます。

### 7.2 参加履歴

これまでに参加表明したチャレンジの一覧を確認できます。過去のイベントへの貢献度も表示されます。

### 7.3 獲得バッジ

活動に応じて獲得したバッジ（アチーブメント）を確認できます。

| バッジ例 | 条件 |
|----------|------|
| 初参加 | 初めてチャレンジに参加 |
| 連続参加 | 3回連続でチャレンジに参加 |
| 招待達人 | 5人以上の友人を招待 |
| 初主催 | 初めてチャレンジを作成 |

## 8. よくある質問

### Q1. ログインできません

**A**: 以下を確認してください。
- インターネット接続が安定しているか
- Xアプリが最新版か
- ブラウザのCookieが有効か

それでも解決しない場合は、アプリを再起動してお試しください。

### Q2. 参加表明を取り消したい

**A**: 現在、参加表明の取り消し機能は提供していません。誤って参加表明した場合は、イベント主催者にお問い合わせください。

### Q3. 友人の追加がうまくいかない

**A**: 友人のXユーザー名（@から始まるID）を正確に入力してください。プロフィールが非公開の場合は、情報を取得できないことがあります。

### Q4. 地図で自分の都道府県が表示されない

**A**: 参加表明時に都道府県を選択していない場合、地図には反映されません。プロフィール設定で都道府県を登録してください。

### Q5. シェアボタンが動作しない

**A**: Web版をご利用の場合、ポップアップブロックが有効になっていると、シェア画面が開かないことがあります。ブラウザの設定でポップアップを許可してください。
