# GitHub Copilot 料金プラン比較（2026年1月時点）

## 概要

GitHub Copilotには3つの個人向けプランがあります。**「$4/月」のプランは存在せず**、実際の料金は以下の通りです。

## 料金プラン一覧

| プラン | 料金 | 対象 |
|--------|------|------|
| **Free** | $0 | 学生・教師・人気OSSメンテナー（要認証） |
| **Pro** | $10/月（または$100/年） | 個人開発者 |
| **Pro+** | $39/月（または$390/年） | ヘビーユーザー |

## Copilot CLI の利用可否

| 機能 | Free | Pro ($10/月) | Pro+ ($39/月) |
|------|------|--------------|---------------|
| **Copilot CLI** | ❌ 利用不可 | ✅ 利用可能 | ✅ 利用可能 |
| ターミナルからCopilot使用 | ❌ | ✅ | ✅ |
| エージェントへのタスク委譲 | ❌ | ✅ | ✅ |
| ローカル/GitHub上でコード変更・コミット | ❌ | ✅ | ✅ |
| プログラマティックモード | ❌ | ✅ | ✅ |

## 主要機能比較

### コード補完

| 機能 | Free | Pro | Pro+ |
|------|------|-----|------|
| コード補完回数 | 2,000回/月 | 無制限 | 無制限 |
| エージェントモード/チャット | 50回/月 | 無制限（GPT-5 mini） | 無制限 |

### プレミアムリクエスト（最新モデル使用）

| 機能 | Free | Pro | Pro+ |
|------|------|-----|------|
| プレミアムリクエスト | 50回/月 | 300回/月 | 1,500回/月 |
| 追加購入 | ❌ | ✅ $0.04/リクエスト | ✅ $0.04/リクエスト |

**プレミアムリクエストを使う機能**:
- チャット
- エージェントモード
- コードレビュー
- コーディングエージェント
- **Copilot CLI**

### 利用可能なAIモデル

| プラン | 利用可能モデル |
|--------|----------------|
| **Free** | Haiku 4.5, GPT-4.1 など基本モデル |
| **Pro** | Anthropic Claude, Google Gemini, OpenAI GPT-5系 など |
| **Pro+** | 全モデル（Claude Opus 4.1含む） |

### その他の機能

| 機能 | Free | Pro | Pro+ |
|------|------|-----|------|
| コーディングエージェント | ❌ | ✅ | ✅ |
| コードレビュー（PR/diff） | ❌ | ✅ | ✅ |
| カスタム指示（instructions.md） | ❌ | ✅ | ✅ |
| GitHub Spark | ❌ | ❌ | ✅ |
| Codex IDE拡張（VS Code） | ❌ | ❌ | ✅ |

## 結論：Copilot CLIを使うには

**Copilot CLIを使用するには、最低でもPro（$10/月）への加入が必要です。**

Freeプランでは以下が利用できません：
- ❌ Copilot CLI
- ❌ コーディングエージェント
- ❌ コードレビュー機能
- ❌ カスタム指示

## Phase 2レビューへの影響

現在のプロジェクトでは、Copilot CLIによる自動レビューは**有料プラン（$10/月以上）が必要**です。

代替案：
1. **手動レビューチェックリスト**（現在採用中）
2. **GitHub Actions + 静的解析ツール**（無料）
3. **ChatGPT/Claude APIによるレビュー**（別途API料金）
