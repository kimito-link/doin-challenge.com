# 手動チェックリスト（1分儀式）

デプロイ後に必ず確認する項目。所要時間：約1分。

---

## ✅ チェック項目

### 1. ホーム画面（15秒）

- [ ] https://doin-challenge.com/ を開く
- [ ] イベント一覧が表示される
- [ ] イベントカードが正しく表示される（画像、タイトル、進捗バー）
- [ ] スケルトン表示が一瞬で消える（キャッシュが効いている）

**期待値:**
- イベント一覧が1秒以内に表示される
- 画像が正しく読み込まれる
- 進捗バーが動作する

---

### 2. イベント詳細画面（20秒）

- [ ] 任意のイベントをタップ
- [ ] イベント詳細が表示される
- [ ] 参加者数が表示される
- [ ] 地図が表示される（都道府県分布）
- [ ] 進捗バーが表示される
- [ ] 「参加予定（現時点）」ラベルが表示される

**期待値:**
- 詳細画面が1秒以内に表示される
- すべてのデータが正しく表示される
- 地図が正しくレンダリングされる

---

### 3. ランキング画面（10秒）

- [ ] ランキングタブを開く
- [ ] 主催者ランキングが表示される
- [ ] 貢献者ランキングが表示される
- [ ] 「総参加予定数（現時点）」ラベルが表示される

**期待値:**
- ランキングが1秒以内に表示される
- ユーザーアイコンが正しく表示される

---

### 4. ログインフロー（15秒）

- [ ] ログインボタンをタップ
- [ ] Twitter OAuth画面に遷移
- [ ] （既にログイン済みの場合）すぐに戻る
- [ ] マイページが表示される
- [ ] プロフィール情報が正しく表示される

**期待値:**
- OAuth認証が正常に動作する
- セッションが維持されている
- プロフィール画像が正しく表示される

---

## 🚨 エラー時の対応

### イベント一覧が表示されない
1. `/api/health` を確認
2. `/api/health?critical=true` を確認
3. ブラウザのコンソールエラーを確認
4. Vercelのログを確認

### 画像が表示されない
1. ブラウザのネットワークタブを確認
2. 画像URLが正しいか確認
3. CORSエラーがないか確認

### ログインできない
1. Twitter OAuth設定を確認
2. コールバックURLが正しいか確認
3. セッション管理が正常か確認
