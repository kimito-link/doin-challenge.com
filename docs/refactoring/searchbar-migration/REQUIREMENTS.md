# SearchBar → SearchInput 移行 - 要件定義

## 目的
SearchBarコンポーネントを統一SearchInputコンポーネントベースに移行し、コードの統一性と保守性を向上させる。

## 現状の問題
- `features/home/components/SearchBar.tsx`が独自の`TextInput`実装を使用
- デバウンス機能とサジェスト機能が実装されているが、統一コンポーネントを使用していない
- 独自のスタイリング（`homeUI`, `homeText`, `homeFont`）を使用

## 要件

### 機能要件
1. **SearchInputの拡張**
   - サジェスト機能を追加
   - デバウンス機能を統合（オプション）

2. **SearchBarの移行**
   - `SearchInput`ベースに変更
   - 既存の機能（デバウンス、サジェスト）を維持
   - スタイリングを統一テーマに変更

### 非機能要件
- 既存の動作を維持
- パフォーマンスへの影響を最小限に
- テーマの統一

## 成功基準
1. SearchBarがSearchInputベースで動作する
2. サジェスト機能が正常に動作する
3. デバウンス機能が正常に動作する
4. スタイリングが統一テーマを使用する
