# キャラクター表示ルール（絶対厳守）

## 基本原則

**りんくは動員ちゃれんじの主人公（メインキャラクター）であり、すべての主要UIで必ず主役として表示される。**

**こん太とたぬ姉は補助的な役割を持ち、特定の場面で適切に使用される。**

## キャラクターの役割

### りんく（主役）

- すべての主要UIで主役として表示
- ログイン画面・モーダル: 常にりんくを表示（優先）
- ホーム画面: メインキャラクターとして表示
- エラーメッセージ、成功メッセージ: りんくを使用

### こん太（相棒・盛り上げ役）

- 友達を誘う・シェア・拡散の場面で使用
- コミュニティ・参加者を増やす場面で使用
- オンボーディングの「友達と一緒に」スライド
- ログインメッセージ: 「一緒に推しを応援しよう！」など

### たぬ姉（仲間・計画役）

- チャレンジ作成・目標設定の場面で使用
- 計画立案・進捗管理の場面で使用
- オンボーディングの「全国マップ」スライド
- ログインメッセージ: 「ログインすると、あなたの応援が記録されるよ！」など

## 視認性のルール

- 背景色とテキスト色のコントラスト比: WCAG 2.1 AA基準（4.5:1以上）
- ダークテーマで視認性が確保されている
- 吹き出しは `colors.surface` / `colors.foreground` を使用し、`colors.border` で境界を明確にする

## チェックリスト

- [ ] ログインモーダルでりんくが優先的に表示される
- [ ] こん太はコミュニティ・盛り上げの場面でのみ使用される
- [ ] たぬ姉は計画・目標設定の場面でのみ使用される
- [ ] ExperienceBannerなどホームの主要UIではりんくを表示
- [ ] 吹き出しのコントラスト比がWCAG 2.1 AAを満たす
