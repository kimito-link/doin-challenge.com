# カラーパレット候補

動員チャレンジアプリに適したパレットを選定。

## 選定基準
- ダークモード対応（背景が暗い）
- 高コントラスト（WCAG AA準拠: 4.5:1以上）
- エネルギッシュで動員・応援のイメージに合う
- アクセントカラーが目立つ

---

## 推奨: 候補3（Refreshing Summer Fun）

**理由**:
1. **ダークティール背景（#023047）** - 目に優しく、長時間使用に適している
2. **オレンジ（#FB8500）** - 動員・応援のエネルギーを表現
3. **イエロー（#FFB703）** - 目標達成の喜びを表現
4. **ライトブルー（#8ECAE6）** - テキストの視認性が高い
5. **コントラスト比** - WCAG AA準拠を満たしやすい
